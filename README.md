# Final Project and Deployment

# AquaFusion 🌊🚗  
**Waterless Mobile Car Wash**

Live Demo 👉 [https://aquafusion.netlify.app/](https://aquafusion.netlify.app/)

## 📌 Overview
AquaFusion is an eco-conscious mobile car wash service that uses waterless car care technology to deliver premium vehicle detailing. This multi-page website was built to showcase AquaFusion’s mission, services, and contact options in a sleek and modern way.

## 🖥️ Pages
- **Home** – Brief intro and CTA
- **About** – Meet the founders and learn the story behind AquaFusion
- **Services** – Overview of services with images and animations
- **Contact** – Contact form and map location

## 🎨 Features
- Responsive design
- Hover effects and animations
- CSS grid & flexbox layouts
- Custom fonts and color palette
- Smooth user experience
- Contact form with interactive UI
- Embedded map (image placeholder)

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (for form and animations)
- [Google Fonts](https://fonts.google.com/)
- Hosted on [Netlify](https://www.netlify.com/)
